# 斗地主
斗地主, golang 1.12, sqlite3, go module

运行步骤:

1.下载源码:

    go get github.com/dwg255/landlord

2.编译:
    
    cd landlord/main
    go build -o ../main.exe
    cd ..
    main.exe

2.启动:
    浏览器访问 http://localhost
